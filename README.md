# KIT-Scrivener-Template
Nach Vorgaben des Department für Geschichte.

Die Datei **KIT-Scrivener-Vorlage.scrivtemplate** enthält das vollständige Template.
Die Datei **KIT Karlsruhe Vorlage - 1.scrformat** enthält das Kompilier-Format.

Die zweite Datei ist in der Datei KIT-Scrivener-Vorlage.scrivtemplate bereits enthalten und muss normalerweise nicht einzeln heruntergeladen werden.
